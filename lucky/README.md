# 班级抽奖网页小工具（可离线）

这是一个纯静态网页（HTML/CSS/JS），可以直接部署到 **Cloudflare Pages**。

## 功能

- 一个大按钮抽奖，带滚动动画
- 按钮位于屏幕中下部，触摸屏更容易点击
- 奖品来自 `coupon.pdf`
- 库存控制（避免抽超）
- 券列表默认收起，可展开查看全部
- 学号输入（可选）+ 数字虚拟键盘
- 本地记录每轮抽奖：轮次、时间、学号、奖品
- 记录可导出 CSV
- 支持离线使用（Service Worker 缓存）

## 本地运行

方式 1：直接双击 `index.html` 打开（部分浏览器离线缓存能力会受限）

方式 2（推荐）：用任意静态服务器打开

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## Cloudflare Pages 部署（最简单）

1. 把当前目录上传到 GitHub 仓库
2. 在 Cloudflare Pages 新建项目，连接这个仓库
3. 构建设置：
   - **Framework preset**: None
   - **Build command**: 留空
   - **Build output directory**: `/`（根目录）
4. 点击部署

> 这是纯静态站点，不需要 Node 构建步骤。

## 如何调整每种券的总数

在 `app.js` 顶部找到：

```js
const DEFAULT_STOCK_PER_COUPON = 2;
```

把 `2` 改成你想要的值（比如 `1`、`3`、`5`）。

### 注意

- 改完后，旧浏览器本地数据不会自动重算。
- 需要点击页面中的 **“重置库存与记录”**，让新库存生效。

## 数据存储说明

- 使用浏览器 `localStorage`
- 存储在当前设备、当前浏览器中
- 换设备/清浏览器数据会丢失
- 建议每次活动后导出 CSV 备份

## 记录字段

- 轮次
- 时间（本地时间）
- 学生姓名
- 奖品名称
- 奖品说明（在导出的 CSV 中）
